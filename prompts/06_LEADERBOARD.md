# Task: Implement Leaderboard Module

## Objective

Implement the complete Leaderboard module for PickTheWinner.

The Leaderboard module is responsible for displaying contestant rankings, tournament standings, prediction statistics and historical performance.

The Leaderboard MUST NOT calculate scores.

The Leaderboard consumes data generated by the Scoring Engine.

The Leaderboard updates automatically whenever match results are published and scores are recalculated.

# Ranking Rules

Ranking is determined by

Total Points

If tied

Apply Tournament Tie-Breaker configuration.

Possible tie-breakers

Most Correct Winners 

One fixed point for normal time+extra time prediction (eg: 5), predicting the winner in the penality will get additional points (eg: 3)

Most Exact Scores

Prediction Accuracy

Earliest Prediction Submitted

Alphabetical Order

Tie-breakers must be configurable.

Do NOT hardcode ranking logic.
